## 관련 코드 파일

| 파일 | 역할 |
|------|------|
| `Scripts/Editor/WaypointDataGenerator.cs` | 웨이포인트 ID, fallbackPosition, radius, locationName |
| `Scripts/Editor/MissionDataGenerator.cs` | POI 생성 (영문/한글명, 타입), 미션-POI 매핑 |
| `Scripts/Core/LocalizationTable.cs` | UI 표시용 다국어 문자열 (장소명 포함 시) |
| `Assets/Data/POIs/RouteA/`, `RouteB/` | 생성된 POI ScriptableObject 에셋 |
| `Assets/Data/Missions/` | 생성된 미션 ScriptableObject 에셋 |
| `docs/웨이포인트_매핑_가이드.md` | 현장 매핑 절차 및 좌표 |
